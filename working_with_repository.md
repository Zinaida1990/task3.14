## git clone 

**git clone *[файл]*** - команда создает копию существующего репозитория.

```bash-
git clone .
```  

## git init 

**git init *[файл]*** - команда  инициализирует новый репозиторий GIT и начинает отслеживание существующего каталога.

```bash-
git init .
```
